Microservicio DevOps funcionando

Grupo: John Zapata y Constanza Mena Aldana

Estado del servicio: ACTIVO

Este microservicio fue desarrollado con Flask e implementa un flujo DevOps completo, utilizando Docker para la contenerización, GitHub Actions para la automatización del pipeline CI/CD, PyTest para las pruebas automatizadas, Trivy para el análisis de vulnerabilidades y Amazon Web Services (AWS) para el despliegue y monitoreo mediante EC2 y CloudWatch.

Herramientas DevOps utilizadas

  • Python 3 y Flask para el desarrollo del microservicio.
  • Docker para la contenerización de la aplicación.
  • Git y GitHub para el control de versiones.
  • GitHub Actions para la automatización del pipeline CI/CD.
  • PyTest para la ejecución de pruebas automatizadas.
  • Trivy para el análisis de vulnerabilidades de la imagen Docker.
  • Amazon EC2 para el despliegue del microservicio.
  • Amazon CloudWatch para el monitoreo de la infraestructura.
